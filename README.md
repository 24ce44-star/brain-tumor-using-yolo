#  Brain Tumor Detection and Size Estimation using YOLO

![image alt](https://github.com/24ce44-star/brain-tumor-using-yolo/blob/77db2e446265631db265cb0e20725661d6d24ca5/Screenshot%202026-01-08%20105153.png)

This project is a **Streamlit-based web application** that detects **brain tumors from MRI images** using a **YOLO object detection model** and estimates the **tumor size in millimeters**.

The application allows users to upload MRI images or run detection on a folder of images, visualizing tumor bounding boxes along with real-world size measurements.


##  Features

-  Brain tumor detection using YOLO
-  Bounding box visualization on MRI images
-  Tumor size estimation in **millimeters (mm)**
-  Upload single MRI images
-  Run detection on a validation/test folder
-  Supports multiple tumor detections per image
-  Runs on CPU (GPU optional)
-  Interactive Streamlit web interface

## Tech Stack

- **Python**
- **YOLO (Ultralytics)**
- **OpenCV**
- **Streamlit**
- **NumPy**
- **Pillow**
- **PyTorch**
